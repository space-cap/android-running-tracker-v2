
러닝 트래커 앱

1. 시작 버튼: 클릭 시 포그라운드 서비스 시작 + 상단 알림창에 "운동 중" 표시.
2. 실시간 경로: 서비스에서 수집한 위도/경도 좌표를 화면의 지도 위에 Polyline으로 실시간 선 그리기.
3. 화면 유지: 운동 중에는 화면이 꺼지지 않게 처리하고, 앱이 백그라운드로 가도 위치 수집이 멈추지 않아야 함. 운동 일시정지 시에는 화면 유지를 해제
4. 결과 저장: 운동 종료 후 이동 거리, 시간 등을 계산하여 로컬 DB(Room)에 저장.
5. 데이터 유실 방지 : 서비스가 시스템에 의해 강제 종료되었다가 복구될 때(START_STICKY), 데이터 유지 전략 필요
6. GPS 꺼짐 대응: GPS 연결이 안 될 경우의 알림 처리
