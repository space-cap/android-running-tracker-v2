# 프로젝트 규칙 (Project Rules)

이 파일은 프로젝트 진행 시 준수해야 할 규칙들을 정의합니다.

## 1. 언어 (Language)
- **모든 문서**(기획서, 결과물, 주석 등)와 **대화**는 **한국어(Korean)**로 작성합니다.
- 단, 코드 내의 변수명, 함수명 등은 영어를 사용합니다.

## 2. 문서화 (Documentation)
- 새로운 기능을 구현할 때는 `docs/` 폴더 내에 계획서(`.md`)를 먼저 작성하고 진행합니다.
- 작업이 완료되면 `walkthrough.md`에 결과를 요약합니다.
