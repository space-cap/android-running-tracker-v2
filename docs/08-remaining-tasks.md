# Running Tracker V2 - 남은 작업 목록 (Remaining Tasks)

현재 프로젝트의 핵심 기능(위치 추적, 서비스, 기본 UI)은 구현되었으나, 앱의 완성도를 높이기 위해 필요한 남은 작업들을 정리합니다.

## 1. 데이터 영속성 및 자동화 (Persistence & Automation)
- [x] **HomeViewModel 도입**: UI 로직과 데이터 로직 분리 완료.
- [x] **기본 저장 로직 구현**: 러닝 종료 시 시간, 거리, 속도 등의 데이터를 Room DB에 저장 기능 완료.
- [x] **알림 클릭 시 앱 복귀**: 알림을 통해 앱으로 돌아오는 PendingIntent 구현 완료.
- [x] **지도 스냅샷 자동 캡처**: 러닝 종료 시점의 경로가 포함된 지도를 Bitmap으로 캡처하여 `RunRecord`와 함께 저장 완료.

## 2. 화면 추가 및 내비게이션 (Navigation)
- [x] **내비게이션 구조 설계**: `MainActivity`에 `NavHost`를 도입하여 화면 간 이동 관리 완료.
- [x] **러닝 기록 목록 화면 (Run List Screen)**: 저장된 러닝 기록을 리스트 형태로 조회 기능 완료.
- [ ] **성공적인 러닝 결과 화면**: 방금 끝낸 러닝의 요약을 보여주는 결과 화면 추가.
- [ ] **기록 상세 화면 (Run Detail Screen)**: 특정 기록의 경로와 통계 상세 보기.
- [ ] **통계 화면 (Statistics Screen)**: 주간/월간 단위의 총 거리 등 요약 대시보드.

## 3. UI/UX 및 편의 기능 (Refinement)

- [ ] **일시 중지/재개 기능**: 운동 중 잠시 멈출 수 있는 기능 추가.
